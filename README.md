# Flux - 高性能移动端 Web 充电解决方案 (PWA)

Flux 是一款基于 **React 19** 构建的渐进式 Web 应用 (PWA)，专为移动端浏览器优化。它旨在为 iOS、Android 及鸿蒙系统用户提供一致、流畅且高效的电动汽车充电体验，无需下载安装即可享受类原生性能。

## 🌟 核心特性
*   **Web 全兼容**: 适配 iOS Safari、Android Chrome 及鸿蒙原生浏览器。
*   **PWA 体验**: 支持“添加到主屏幕”，具备沉浸式全屏交互。
*   **实时监控**: 利用模拟 WebSocket 技术实现充电数据动态推送。
*   **安全验证**: 基于浏览器标准的 WebAuthn 生物识别验证。
